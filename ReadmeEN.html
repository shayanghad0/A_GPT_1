About the Project

The AGPT Project is a sophisticated, lightweight web application designed to bridge the gap between users and powerful Large Language Models (LLMs). By utilizing the Flask framework and OpenAI's API architecture, it provides a real-time, interactive chat experience directly through your web browser.

Getting Started

Follow these steps to get your local environment running smoothly.

1. Prerequisites

  • Python 3.10 or higher installed.
  • pip installed (usually comes with Python).

2. Installation & Deployment

# Step 1: Clone the repository
git clone https://github.cAmirprogrammerA/A_gpt.git
cd A_gpt

# Step 2: Set up a Virtual Environment (Highly Recommended)
# On Windows:
python -m venv venv
venv\Scripts\activate

# On macOS/Linux:
python3 -m venv venv
source venv/bin/activate

# Step 3: Install necessary libraries
pip install flask openai

3. Running the Application

python app.py

Once the server is active, navigate to: http://127.0.0.1:5320

Troubleshooting

Common issues and their respective solutions:

Issue Solution
ModuleNotFoundError You missed a library. Run pip install flask openai.
pip not recognized Try using python -m pip install ... instead.
No response from AI Check your internet connection and ensure the API endpoint in app.py is reachable.
Port already in use Change the port in app.py from 5320 to another number like 5000.

Frequently Asked Questions (FAQ)

Q: Why should I use a Virtual Environment?

A: It keeps your project dependencies isolated from your global Python installation, preventing version conflicts with other projects.

Q: Can I use this project for Persian language?

A: Yes! The backend is designed to process and respond to multiple languages, including Persian, seamlessly.

Q: Do I need an OpenAI API key?

A: Yes, the application interacts with an AI endpoint. Ensure your app.py is configured with a valid API key/endpoint.
