## 📌 Overview

**AGPT Project** is  streamlined, web-based AI chat interface designed for immediate use. Unlike most AI applications, this project requires **zero manual configuration**. It comes pre-configured with a specialized API endpoint, allowing users to interact with the powerful **GPT-4o** model instantly without needing to manage their own API keys or environment variables.

| Feature | Description |

## 🚀 Getting Started

Since this project is pre-configure, you can get it running in seconds!

### 1️⃣ Prerequisites
- **Python 3.10+**
- **pip** (Python package manager)

### 2️⃣ Installation
```bash
# 1. Clone the repository
git clone https://github.com/AmirprogrammerA/A_GPT_1.git

# 2. Navigate into the project folder
cd A_GPT_1

# 3. Install required dependencies
pip install flask openai
-----------------------

3️⃣ Running the App


python app.py
Once started, open your browser and go to:

👉 http://127.0.0.1:5000
